## Команды для командной строки:

ls - просмотр содержимого папки

pwd - просмотр пути к папке в которой находишься

touch - создать файл

mkdir - создать папку

cp - копировать объект

mv - переместить объект (я не понял, как это работает)

cat - прочитать файл

rm - удалить файл

rmdir - удалить папку (только пустую)

rm -r - удалить папку вместе с её содержимым
